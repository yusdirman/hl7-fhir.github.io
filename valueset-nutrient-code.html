Name:Nutrient Modifier Codes
Definition:NutrientModifier : Codes for types of nutrient that is being modified such as carbohydrate or sodium. This value set includes all the children of SNOMED CT Concepts from SCTID 226355009 Nutrients (substance), Sodium, Potassium and Fluid and is provided as a suggestive example.
