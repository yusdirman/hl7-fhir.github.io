D.23.0 DAF FHIR IG Supporting Profiles

This document provides links to the various supporting profiles within the context of the DAF FHIR IG. The supporting profiles are defined as those which are used within other DAF profiles. DAF use cases will not be querying the resources represented by the supporting profiles directly. For e.g DAF Organization profile and DAF Practitioner profiles are used in other DAF profiles of resources such as Patient, Medication Administration, Condition etc.


DAF Supporting profile Purpose Base FHIR Resource
DAFPractitioner This profile is used to represent a medical practitioner who delivers various clinical services. Practitioner
DAFOrganization This profile is used to represent an organization where clinical services or delivered or is somehow related to the treatment of the patient. Organization
DAFLocation This profile is used to represent a location associated with the delivery of clinical services to a patient. Location
DAFSubstance This profile is used to represent either a substance which is part of a drug or is a substance which is known to cause allergies to a patient. Substance
DAFSpecimen This profile is used to represent a specimen that is used to perform lab tests or other clincal observations against. Specimen
DAFRelatedPerson This profile is used to represent a person who is related to a patient. RelatedPerson
