1.30.2.1.418 Value Set http://hl7.org/fhir/ValueSet/procedure-not-performed-reason

This is a value set defined at http://hl7.org/special/committees/CQI .

Summary

Defining URL:http://hl7.org/fhir/ValueSet/procedure-not-performed-reason
Name:Procedure Not Performed Reason (SNOMED-CT)
Definition:Situation codes describing the reason that a procedure, which might otherwise be expected, was not performed, or a procedure that was started and was not completed. Consists of SNOMED CT codes, children of procedure contraindicated (183932001), procedure discontinued (416406003), procedure not done (416237000), procedure not indicated (428119001), procedure not offered (416064006), procedure not wanted (416432009), procedure refused (183944003), and procedure stopped (394908001).
Committee:Patient Care Work Group
OID:2.16.840.1.113883.4.642.2.418 (for OID based terminology systems)
