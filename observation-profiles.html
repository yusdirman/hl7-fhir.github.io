4.31.10 Resource Observation - Profiles

FHIR Specification Core
Device Metric Observation This profile describes the direct or derived, qualitative or quantitative physiological measurement, setting, or calculation data produced by a medical device or a device component.
Observation HL7 Extensions Defines common extensions used with or related to the Observation resource
Observation for Genetics Describes how the observation resource is used to report structured genetic test results
Vital Signs Profile Defines constraints and extensions on the Observation resource for use in querying and retrieving Vital Signs information.
US Lab Implementation Guides
US Laboratory Observation Profile Constraints and extensions to the Observation resource for the US Realm laboratory ordering and resulting in ambulatory care setting
Data Access Framework
U.S. Data Access Framework (DAF) Vital Signs Profile Defines constraints and extensions on the Observation resource for use in querying and retrieving Vital Signs information.
U.S. Data Access Framework (DAF) Results Profile Defines constraints and extensions on the Observation resource for use in querying and retrieving Result information.
U.S. Data Access Framework (DAF) Smoking Status Profile Defines constraints and extensions on the Observation resource for use in querying and retrieving Smoking Status information.
QICore Implementation Guide
QICore-Observation Profile of Observation for decision support/quality metrics. Defines the core set of elements and extensions for quality rule and measure authors.
HSPC Implementation Guide
HSPC Standard Lab Observation StandardLabObs is the base Model for standard laboratory test results/observations such as hematology, chemistry, coagulation, serology and blood bank. This model will not be used for microbiology nor anatomic pathology data
HSPC Standard Quantitative Lab Observation StandardLabObs is the base Model for standard laboratory test results/observations such as hematology, chemistry, coagulation, serology and blood bank. This model will not be used for microbiology, and anatomic pathology data
HSPC Age Profile The HSPC specific profile for the observation of an indivudual's age.
HSPC Height Profile The HSPC specific profile for the observation of one's height or body length.
HSPC Standard Coded Lab Observation StandardLabObsCoded is the model for laboratory test results that have coded values.
HSPC Standard IVLPQ Lab Observation Standard IVLPQ lab observation is the model for laboratory tests that have interval physical quantity values.
HSPC Standard Narrative Lab Observation StandardLabObsNarrative is the model for laboratory test results that have narrative or textual values.

HSPC Standard Titer Lab Observation StandardLabObsTiter is the model for laboratory test results that have titration results
HSPC Standard Ordinal Lab Observation Standard Ordinal Lab Observation is the model for laboratory test results that use ordinal values such as 1+, 2+, Positive, Negative, etc.
HSPC Body Weight Profile The HSPC specific profile for the observation of one's body weight.
HSPC Heart Rate Profile The HSPC specific profile for the observation of the speed at which one's heart is beating.
HSPC Head Occipital Frontal Circumference Profile The HSPC specific profile for the observation of the occipital-frontal head circumference.
HSPC Estimated Due Date Profile The HSPC specific profile for the observation of an estimated pregnancy due date
HSPC Bone Age Profile The HSPC specific profile for a bone age via x-ray observation
HSPC Body Mass Index The HSPC specific profile for a body mass index observation
HSPC Respiratory Rate Profile The HSPC specific profile for the observation of the rate at which one is breathing.
HSPC Gestational Age at Birth The HSPC specific profile for the observation of the length of time from conception to birth.
HSPC Body Temperature The HSPC specific profile for a body temperature observation
HSPC Diastolic Blood Pressure The HSPC specific profile for the observation of the diastolic blood pressure.
HSPC Systolic Blood Pressure The HSPC specific profile for the observation of the systolic blood pressure.
HSPC Blood Pressure Panel The HSPC specific profile for the observation of a blood pressure panel.
HSPC Mean Arterial Pressure The HSPC specific profile for the measurement of a mean arterial pressure.
HSPC Logical Assert The HSPC specific observation profile for logical assertions, these are observations where the value is a logical conclusion from a process.
