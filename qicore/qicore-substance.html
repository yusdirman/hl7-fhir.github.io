F.35.1 StructureDefinition: QICore-Substance

The official URL for this profile is: 

http://hl7.org/fhir/StructureDefinition/qicore-substance

Profile of Substance for decision support/quality metrics. Defines the core set of elements and extensions for quality rule and measure authors.

This profile was published on Tue, Mar 17, 2015 00:00+0000 as a draft by Health Level Seven International (Clinical Quality Information - QICore).
