F.5.1 StructureDefinition: QICore-Communication

The official URL for this profile is: 

http://hl7.org/fhir/StructureDefinition/qicore-communication

Profile of Communication for decision support/quality metrics. Defines the core set of elements and extensions for quality rule and measure authors.

This profile was published on Fri, Feb 27, 2015 00:00+0000 as a draft by Health Level Seven International (Clinical Quality Information - QICore).
