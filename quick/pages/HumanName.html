HumanName

  • Name of a human - parts and usage

    Base StructureDefinition for HumanName Type

    • Key:  = Must support,  = QICore-defined extension,  = Is-Modifier

       Field Card. Type and Description
        family 0..* List<String>
      The part of a name that links to the genealogy. In some cultures (e.g. Eritrea) the family name of a son is the first name of his father.
        given 0..* List<String>
      Given name.
        period 0..1 interval&lt;DateTime&gt;
      Indicates the period of time when this name was valid for the named person.
        prefix 0..* List<String>
      Part of the name that is acquired as a title due to academic, legal, employment or nobility status, etc. and that appears at the start of the name.
        suffix 0..* List<String>
      Part of the name that is acquired as a title due to academic, legal, employment or nobility status, etc. and that appears at the end of the name.
        use 0..1 code
      Identifies the purpose for this name.
