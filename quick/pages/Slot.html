Slot

  • A slot of time on a schedule that may be available for booking appointments

    Base StructureDefinition for Slot Resource

    • Key:  = Must support,  = QICore-defined extension,  = Is-Modifier

       Field Card. Type and Description
        comment 0..1 String
      Comments on the slot to describe any extended information. Such as custom constraints on the slot.
        end 1..1 DateTime
      Date/Time that the slot is to conclude.
        freeBusyType 1..1 code
      busy | free | busy-unavailable | busy-tentative.

      Binding: SlotStatus (Required)
        identifier 0..* List<Identifier>
      External Ids for this item.
        overbooked 0..1 Boolean
      This slot has already been overbooked, appointments are unlikely to be accepted for this time.
        schedule 1..1 Schedule
      The schedule resource that this slot defines an interval of status information.
        start 1..1 DateTime
      Date/Time that the slot is to begin.
        type 0..1 Concept
      The type of appointments that can be booked into this slot (ideally this would be an identifiable service - which is at a location, rather than the location itself). If provided then this overrides the value provided on the availability resource.
