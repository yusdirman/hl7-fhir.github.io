This value set includes codes from the following code systems:

  • Include these codes as defined in http://hl7.org/fhir/v3/EntityNameUse
    CodeDisplay
    LLegal NameKnown as/conventional/the one you use
    ASGNAssigned (alias)A name assigned to a person. Reasons some organizations assign alternate names may include not knowing the person's name, or to maintain anonymity. Some, but not necessarily all, of the name types that people call "alias" may fit into this category.
  • Include these codes as defined in http://hl7.org/fhir/v3/EntityNameUseR2
    CodeDisplay
    Mmaiden nameDescription:A name used prior to marriage.Note that marriage naming customs vary greatly around the world. This name use is for use by applications that collect and store maiden names. Though the concept of maiden name is often gender specific, the use of this term is not gender specific. The use of this term does not imply any particular history for a person's name, nor should the maiden name be determined algorithmically
