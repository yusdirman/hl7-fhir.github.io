Extensions:
hspc-extensionsdataOriginationModeData Origination Mode :

The means by which the data came into being.

hspc-extensionsassociatedPreconditionAssociated Precondition :

Conditions that exist during measurement.

hspc-extensionsalleviatingFactoralleviatingFactor :

Actions taken that reduce or resolve a condition or problem.

hspc-extensionsexacerbatingFactorexacerbatingFactor :

Actions or influences that increase the severity of a condition or problem.

hspc-extensionsperiodicityperiodicity :

How often or specific time frames in which a condition exists.

hspc-extensionsassociatedSignAndSymptomassociatedSignAndSymptom :

Conditions or problems that are related to the asserted condition or problem.

hspc-extensionscoursecourse :

The developemental sequence for a condition or problem.

hspc-extensionsdeltaFlagDelta Flag :

The qualitative change in the value relative to the previous measurement. Usually only recorded if the change is clinically significant.

hspc-extensionsbodyPositionbodyPosition :

The position of the body when the observation was done, e.g. standing, sitting. To be used only when the body position in not precoordinated in the observation code.

hspc-extensionsfocalSubjectFocal Subject :

The focus of an observation when not the patient of record.

hspc-extensionsbodyLateralityBody Laterality :

The set of concepts that describe the side of the body an observation refers to.

hspc-extensionshistoryIndicatorHistory Indicator :

An indication that the patient has a history of the clincal assertion being made and may or may not currently be exibiting the condition.

hspc-extensionslikelihoodLikelihood :

A code that identifies the data that defines the level of surety of the logical assertion.

hspc-extensionsriskForIndicatorRisk for Indicator :

An indication that there is a probability to have the stated condition.

hspc-extensionsdateOfResolutionDate of Resolution :

The date that a allergy or intolerance was resolved.

hspc-extensionspatientInformedIndPatient Informed Indicator :

An indication that the patient was given information.

hspc-extensionscauseOfDeathIndCause of Death Indicator :

An indicator that the allergy or intolerance was the cause of death.

hspc-extensionssupportingInformationIdentifierSupportingInformationId :

An identifier for the data that supports the logical assertion.

hspc-extensionssupportingInformationTextSupportingInformationText :

A textual representation of the data that supports the logical assertion.

hspc-extensionsknowledgeDocumentIdentifierKnowledgeDocumentId :

A code that identifies a document that contains information about the logical assertion.

hspc-extensionsdateOfOnsetDate of Onset :

The date that an asserted condition began.

hspc-extensionscredentialEligibilityIndCredential Eligibility Indicator :

An indication that the individual is in procedings or eligible to obtain credentials.

hspc-extensionscommunicationModeCommunicationMode :

The method one uses to communicate.

hspc-extensionspracticeAffiliationTypeAffiliationType :

The kind of practice affilation a provider has with the organization.

hspc-extensionspractitionerPrivilegePractitionerPrivilege :

The level at which a provider is allowed to practice at a specific institution.

hspc-extensionsprimaryIndicatorA flag indicating prime :

This indicatest that this is the principal address.

hspc-extensionsaddressTypeThe type of address :

The kind of address, e.g. home, work, etc.

hspc-extensionsdeceasedDateTimeDeceased DateTime :

hspc-extensionspreferenceIndicatorPreference Ind :

An indication that the stated value is the preferred value, i.e. address, phone number, etc.

hspc-extensionsconfidentialityConfidentiality :

A set of codes that describes the individuals allowed to view the given data.

hspc-extensionspriorityPriority :

A set of codes that describe the urgency level.

hspc-extensionsreportingPriorityReporting Priority :

A set of codes that describe the urgency at which the report should be given to the clinician.

hspc-extensionsresultCopiesToResult Copies To :

The results need to be copied and given to the provider indicated.

hspc-extensionsradiologyTransportModeTransport mode :

The set of codes that describe the different means used to get a patient to the diagnostic imaging department.
